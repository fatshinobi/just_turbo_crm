# Just Turbo CRM
Training CRM application that based on Turbo (and little Stimulus)

## Application Description
The application based on [Just CRM](https://github.com/fatshinobi/justcrm) business logic


## System Requirements
  * Ruby 3.0
  * Ruby On Rails 7.0
